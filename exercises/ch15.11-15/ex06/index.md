## localStorage と sessionStorage それぞれに保存されたデータの有効期限がどのように異なるか、実際に動作確認して結果を記述しなさい。

### 動作確認と結果

| 動作確認                                 | localStorage | sessionStorage |
| ---------------------------------------- | ------------ | -------------- |
| ページリロード                           | データは残る | データは残る   |
| ブラウザを閉じて再度開く                 | データは残る | データは消える |
| 別タブを開く                             | データは残る | データは消える |
| タブを閉じて再度開く                     | データは残る | データは消える |
| シークレットモードでタブを閉じて再度開く | データは残る | データは消える |
| シークレットモードで別タブを開く         | データは残る | データは消える |
