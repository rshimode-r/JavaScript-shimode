# Web アプリケーションがユーザの機密情報をセキュアに扱うためにはどのようなことが必要になるか記述しなさい。ブラウザでの対応に限らず、Web アプリケーション全般を対象として考えること。

### クライアント

- localStorage や sessionStorage に平文で機密情報を保存しない
- どうしても必要な場合は、暗号化したものを保存する
- Cookie は HttpOnly、Secure、SameSite を設定して盗難や CSRF を防ぐ

### サーバー

- 機密情報はサーバー側で安全にデータ保存する必要がある。
- 個人情報や機密データは、AES などの安全とされている暗号アルゴリズムを用いて暗号化して保存する

### 通信

- HTTPS を利用してクライアントとサーバー間の通信を保護する
- 中間者攻撃や盗聴を防ぐ
